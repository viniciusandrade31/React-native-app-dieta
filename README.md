Controle Alimentar App

Sobre o projeto
Aplicativo mobile desenvolvido com React Native (Expo) e TypeScript com o objetivo de ajudar no controle alimentar diário.

A aplicação permite registrar alimentos, acompanhar refeições e organizar informações de forma simples e visual.

Objetivo
Este projeto foi desenvolvido para praticar:

- React Native com Expo
- Organização de projeto
- Componentização
- Uso de TypeScript
- Estruturação de telas e lógica básica

Tecnologias utilizadas
- React Native
- Expo
- TypeScript
- JavaScript

Funcionalidades
- Cadastro de alimentos/refeições
- Visualização das refeições
- Organização das informações na interface
- Manipulação de dados localmente
